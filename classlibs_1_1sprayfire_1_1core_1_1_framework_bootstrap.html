Detailed Description

The framework's default bootstrap, is primarily responsible for writing the default values in the CoreConfiguration object and running each bootstrap object located in the app/bootstrap folder.

libs\sprayfire\core\FrameworkBootstrap::getAppBootstrapObjects ( ) [private]

Will return an array of bootstrap objects that properly implement the framework bootstrapper interface.

If no files are found in the app/bootstrap directory or the objects in that directory do not implement the proper interface the array of objects may be an empty array. Either way, an array will be returned.
