Detailed Description

An object that allows for data to be stored and to be assured that the data is not mutable.

This object is immutable by the fact that after the object is constructed attempting to __set the object or offsetSet the object's properties will results in a libs.sprayfire.exceptions.UnsupportedOperationException will be thrown. If a class extends this please ensure that it is a truly immutable object and does not have any "backdoors".

Converts all arrays in $data to ImmutableStorage objects, allowing for the chaining of properties in the created object.

Note that if you extend ImmutableStorage and override this method an array value MUST be returned or a libs.sprayfire.exceptions.UnexpectedValueException will be thrown by the class constructor. If self::__construct() is overridden as well and the data from convertDataDeep is not an array you will receive a type hint compile error when parent::__construct() is called.
