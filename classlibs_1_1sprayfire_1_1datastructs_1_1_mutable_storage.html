libs\sprayfire\datastructs\MutableStorage::convertDataDeep ( array $  data) [protected]

Converts all arrays in $data to MutableStorage objects, allowing for the chaining of properties in the created object.

Note that if you extend MutableStorage and override this method an array value MUST be returned or an UnexpectedValueException will be thrown by the class constructor. If self::__construct() is overridden as well and the data from convertDataDeep is not checked you will receive a type hint compile error if a non-array is passed to parent::__construct().
