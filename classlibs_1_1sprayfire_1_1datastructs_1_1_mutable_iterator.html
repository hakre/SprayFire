Detailed Description

This class is a data storage that also acts as an iterator, allowing objects extending this class to be used in a foreach loop.

This code is heavily influenced by the Zend Framework 2 Zend.Config.Config class. https://github.com/zendframework/zf2/blob/master/library/Zend/Config/Config.php
