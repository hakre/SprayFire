Detailed Description

Will take a libs.sprayfire.config.JsonConfig object to determine the mapped URI to use for the request and convert any libs.sprayfire.request.Uri object passed into an appropriate libs.sprayfire.request.RoutedUri object.

This class **GUARANTEES** that when SprayFireRouter::getRoutedUri($Uri) is invoked an appropriate object will be returned, even if an invalid configuration file is used.

libs\sprayfire\request\SprayFireRouter::__construct ( Configuration $  RoutesConfig)

Will set the appropriate class properties based on the $RoutesConfig object passed.

It is expected that the $RoutesConfig object will have 2 properties set:

$RoutesConfig->defaults->controller

and

$RoutesConfig->defaults->action

Obviously, these values will be used for the $defaultController and $defaultAction attributes if they appear. If they are not present in the $RoutesConfig object an error will be logged and a framework chosen controller and action will be invoked.

libs\sprayfire\request\SprayFireRouter::getRoutedUri ( Uri $  Uri)

Will take a libs.sprayfire.request.Uri object and create an appropriate libs.sprayfire.request.RoutedUri object.

Will produce a SprayFireURIPattern based on the $Uri object passed, determine whether the default controller and action or the controller and action passed in $Uri should be used, it will then produce the appropriate mapped URI string, create a libs.sprayfire.request.RoutedUri, set the original URI string passed and finally return the RoutedUri object created.
